# Void Hauler 🚀

A sci-fi cargo trading CLI game built with Python.

Pilot a cargo ship across star systems, trade goods, upgrade your ship,
and build an interstellar freight empire.

## Tech Stack

- Python
- Typer (CLI)
- Rich (terminal UI)
- Pydantic (data models)

## Features (planned)

- Cargo trading
- Ship upgrades
- Planet economies
- Travel routes
- ASCII sci-fi UI
